# <a name="create-small-multiples-in-power-bi-preview"></a>Создание маленьких диаграмм в Power BI (предварительная версия)

[!INCLUDE [applies-to](../includes/applies-to.md)] [!INCLUDE [yes-desktop](../includes/yes-desktop.md)] [!INCLUDE [yes-service](../includes/yes-service.md)]

Маленькие диаграммы создаются путем разделения визуального элемента на несколько его версий. Версии отображаются рядом друг с другом, при этом данные распределяются между ними по выбранному измерению. Например, маленькая диаграмма может разделить диаграмму со столбцом "продажи по категориям" по линейкам продуктов или регионам. В этой предварительной версии маленькие диаграммы имеют небольшой набор возможностей, но в будущих выпусках мы расширим функциональность.

:::image type="content" source="media/power-bi-visualization-small-multiples/small-mulitple-sales-category-region.png" alt-text="Снимок экрана: маленькие диаграммы для категории и региона.":::

## <a name="enable-the-preview-feature"></a>Включение предварительной версии функции

В меню **Файл** выберите **Параметры и настройки** > **Параметры** > **Функции предварительной версии** и установите флажок **Маленькие диаграммы**.

:::image type="content" source="media/power-bi-visualization-small-multiples/small-multiple-enable-preview.png" alt-text="Включение предварительной версии маленьких диаграмм.":::

Перезапустите Power BI Desktop, чтобы начать работу с маленькими диаграммами.

## <a name="create-small-multiples"></a>Создание маленьких диаграмм

В настоящее время маленькие диаграммы можно создавать для линейчатых диаграмм, гистограмм, графиков и диаграмм с областями. 

Чтобы приступить к работе, создайте один из приведенных выше визуальных элементов и выберите поле, по которому вы хотите секционировать данные. Перетащите это поле в область **Маленькие диаграммы** в разделе "Поля" на панели "Визуализации". Диаграмма разбивается на сетку размером 2×2, а данные распределяются по выбранному измерению. Сетка будет заполнена маленькими диаграммами. Они сортируются в выбранном порядке сортировки слева направо и сверху вниз.

:::image type="content" source="media/power-bi-visualization-small-multiples/small-multiple-two-by-two-grid.png" alt-text="Маленькие диаграммы в сетке 2×2.":::

Вы увидите, что оси синхронизированы. Слева от каждой строки есть одна ось Y, а в нижней части каждого столбца — одна ось X.

Теперь, когда вы создали маленькие диаграммы, узнайте, как [взаимодействовать с ними в Power BI (предварительная версия)](power-bi-visualization-small-multiples-interact.md).

## <a name="format-a-small-multiples-visual"></a>Форматирование элемента маленькой диаграммы

Некоторые новые параметры в области форматирования позволяют менять внешний вид сетки и способ взаимодействия с ней.

### <a name="change-the-grid-dimensions"></a>Изменение размеров сетки

Вы можете изменить размеры сетки в карточке "Макет сетки":

:::image type="content" source="media/power-bi-visualization-small-multiples/small-multiple-grid-layout-card.png" alt-text="Карточка &quot;Макет сетки&quot; с маленькими диаграммами.":::

По умолчанию используется сетка размером 2×2, но можно увеличить число строк и столбцов до 6×6. Все диаграммы, которые не помещаются в такой сетке, будут загружены при прокрутке.


### <a name="adjust-the-small-multiples-titles"></a>Изменение заголовков небольших диаграмм

Как и в случае с заголовками других визуальных элементов, вы можете настроить стиль и расположение заголовков маленьких диаграмм на карточке **Заголовок маленькой диаграммы**.

## <a name="preview-limitations"></a>Ограничения предварительной версии

Хотя эта функция предоставляется в предварительной версии, мы постоянно совершенствуем способ ее взаимодействия с другими функциями. Ниже приведены некоторые действующие сейчас ограничения.

### <a name="fields-pane"></a>Панель "Поля"

- Дата и другие непрерывные иерархии: предположим, что у вас есть визуальный элемент (например, график) с иерархией дат на оси X. При создании из него маленьких диаграмм Power BI преобразует эту ось из непрерывной в ось категорий.
- Отображение элементов без данных: этот параметр еще доступен, но его поведение может не совпадать с вашими ожиданиями.

### <a name="visual-interactions"></a>Взаимодействие визуальных элементов

- Прокрутите, чтобы загрузить больше данных по оси категорий: в стандартных визуальных элементах со множеством категорий на оси при прокрутке до конца оси визуальный элемент загружает дополнительные категории. В настоящее время визуальный элемент маленькой диаграммы не загружает дополнительные категории.
- Сортировка маленьких диаграмм по измерению: сортировка маленьких диаграмм — это новая функция. Возможно, вы хотите сортировать маленькие диаграммы по измерению. В настоящее маленькие диаграммы можно сортировать только в естественном порядке сортировки поля.
- Щелчок правой кнопкой мыши/контекстное меню -> "Анализ": пока отключено.
- Щелчок правой кнопкой мыши/контекстное меню -> "Сводка": пока отключено.
- Выбор нескольких точек данных с прямоугольным выделением: пока отключено.
- Масштаб оси: пока отключено.
- Специальные возможности: вы можете настроить навигацию с помощью клавиатуры и чтение с экрана, чтобы упростить работу с новым уровнем сетки, который используется маленькими диаграммами. Некоторое поведение не реализовано, например навигация с помощью клавиатуры по полосе прокрутки для оси категорий.

### <a name="formatting-options"></a>Параметры форматирования

**Общие сведения**

- Переключатель с быстрым откликом: этот параметр еще доступен, но его поведение может не совпадать с вашими ожиданиями. Так как многие мобильные операции используют этот переключатель, интерфейс на мобильных устройствах очень похож на интерфейс Power BI Desktop и службы Power BI.
- Выборка с высокой плотностью: для графиков переключатель выборки с высокой плотностью еще доступен, но в настоящее время не поддерживается маленькими диаграммами.

**Ось**

- Сцепление меток: пока отключено.

**Метки итогов**

- Общее число меток для диаграмм с накоплением: пока отключено.

**Ползунок масштабирования**

- Ползунки масштабирования: пока отключено.

**Область аналитики** 

- Линии трендов: пока отключено.
- Прогнозирование: пока отключено.

Динамическое форматирование для выделенных меток: в настоящее время не поддерживается.
Доступность службы

## <a name="authoring-in-the-power-bi-service"></a>Разработка в службе Power BI

Разработка маленьких диаграмм в Интернете не поддерживается, так как эта функция предоставляется в предварительной версии. Вы можете просмотреть отчет с визуальным элементом маленьких диаграмм и даже отформатировать его. Но вы не можете преобразовать стандартный визуальный элемент в визуальный элемент маленьких диаграмма в службе Power BI. Визуальный элемент должен уже включать поле в области поля маленьких диаграмм, иначе область маленьких диаграмм не будет отображаться для такого визуального элемента.
