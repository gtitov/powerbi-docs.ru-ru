3. На этапе *Шаг 1. Вход в Power BI* войдите с помощью учетных данных пользователя, который принадлежит вашему клиенту Power BI. Приложение Azure AD будет зарегистрировано для этого пользователя.

    Если вы уже выполнили вход, убедитесь, что вы вошли как пользователь для создания приложения Azure AD. Чтобы изменить пользователя, щелкните ссылку *Выйти* и после перезапуска средства войдите еще раз.
