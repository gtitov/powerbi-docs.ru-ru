6. (Необязательно) На этапе *Шаг 3. Создание рабочей области* можно создать рабочую область в службе Power BI.

    * Если у вас уже есть рабочая область Power BI, щелкните **Пропустить**.

    * Чтобы создать рабочую область, введите имя рабочей области и выберите **Создать рабочую область**. Имя и идентификатор рабочей области отображаются в поле *Сводка*. Скопируйте эти значения для последующего использования.

    >[!NOTE]
    >Чтобы **пример приложения встроенной аналитики** работал должным образом, необходимо создать рабочую область с помощью средства.

7. (Необязательно) На этапе *Шаг 4. Импорт содержимого* выберите один из следующих вариантов.

    * При наличии собственного приложения Power BI можно выбрать **Пропустить**.

    * Если нужно создать пример приложения Power BI с помощью примера отчета, выберите **Пример отчета Power BI**, а затем щелкните **Импорт**.

    * Если нужно создать пример приложения Power BI с помощью собственного отчета, выберите **Отправить PBIX-файл**, а затем щелкните **Импорт**.
