# <a name="power-bi-report-builder-change-log"></a>Журнал изменений построителя отчетов Power BI

[!INCLUDE [applies-to](../includes/applies-to.md)] [!INCLUDE [yes-service](../includes/yes-service.md)] [!INCLUDE [yes-paginated](../includes/yes-paginated.md)] [!INCLUDE [yes-premium](../includes/yes-premium.md)] [!INCLUDE [no-desktop](../includes/no-desktop.md)] 

Сведения о новых элементах, исправлениях ошибок и известных проблемах в каждом выпуске построителя отчетов Power BI начиная с сентября 2020 г. Содержимое этой статьи обновляется с каждым новым выпуском. Новости о других компонентах Power BI см. в статье [Новые возможности в Power BI](../fundamentals/desktop-latest-update.md).

Вы готовы начать работу? [Установите Power BI Report Builder](https://go.microsoft.com/fwlink/?linkid=2086513) из Центра загрузки Майкрософт.

## <a name="january-2021"></a>Январь 2021 г. 

Версия 15.6.01487.0002, дата выпуска: 26 января 2021 г. 

### <a name="bug-fixes"></a>Исправления ошибок 

- Название службы Common Data Service изменено на Dataverse. 

### <a name="features"></a>Компоненты

- Интерфейс для публикации в службе улучшен и согласован с интерфейсом службы Power BI. 
- Добавлены новые значки построителя отчетов Power BI и версии "Premium на пользователя". Цель — сделать фирменное оформление более согласованным, а интерфейс — более современным.

### <a name="known-issues"></a>Известные проблемы

- Ошибка при подключении к рабочей области через конечную точку XMLA в случае, если имя рабочей области содержит знак "плюс" (+).  

### <a name="other"></a>Другое 
Полный список новых и обновленных функций см. в записи блога, посвященной построителю отчетов Power BI, за январь 2021 года.

## <a name="september-2020"></a>Сентябрь 2020 г. 

Версия 15.6.01340.0002, дата выпуска: 3 сентября 2020 г. 

### <a name="bug-fixes"></a>Исправления ошибок

- Исправлена трассировка для лучшего понимания проблем, связанных с интеграцией службы PBI, и проблем на стороне клиента. 
- Исправлена строка подключения во избежание сходства с кластером. 

### <a name="features"></a>Компоненты

- Добавлена поддержка нового типа проверки подлинности для Базы данных SQL Azure и Azure Synapse. Он предусматривает использование многофакторной проверки подлинности Azure AD, как и в службе Power BI.
- Появилась возможность связи с Dataverse через подключение к данным SQL. При этом поддерживается использование собственного конструктора реляционных запросов для SQL. 
- Реализовано автоматическое расширение раскрывающегося списка параметров для отображения даже самого длинного элемента в нем. 
- Улучшена поддержка параметров с несколькими значениями для DAX. 

### <a name="other"></a>Другое 

Полный список новых и обновленных функций см. в записи блога, посвященной построителю отчетов Power BI, за сентябрь 2020 года.
