# <a name="create-quick-reports-in-the-power-bi-service"></a>Создание быстрых отчетов в службе Power BI 

В службе Power BI появился новый способ быстрого создания отчетов. Вместо загрузки приложения Power BI Desktop и импорта данных можно вставить данные прямо в интернет-версию Power BI, и служба автоматически создаст визуальные элементы.  

Не знакомы с возможностями создания в Power BI? Прочтите статью [Отчеты в Power BI](../consumer/end-user-reports.md), чтобы получить общие сведения.

:::image type="content" source="media/service-quick-create-report/quick-create.gif" alt-text="Быстрое создание отчета с помощью Power BI.":::

## <a name="create-a-quick-report"></a>Создание быстрого отчета
Если в области навигации в службе Power BI нажать кнопку **Создать**, откроется страница, на которой можно выбрать источник данных. К ней также можно перейти, нажав кнопку **Создать отчет** на главной странице.

:::image type="content" source="media/service-quick-create-report/create-entry-point.png" alt-text="Значок, позволяющий приступить к созданию в службе Power BI."::: 

В настоящее время поддерживается только создание отчета на основе имеющегося набора данных, а также ввод или вставка данных непосредственно в таблицу. Со временем появится поддержка других источников, например отправка файла Excel.  

:::image type="content" source="media/service-quick-create-report/create-source-options.png" alt-text="Добавление данных для создания отчета, параметры источника.":::

При вводе или вставке данных вы получаете сетку, в которой можно начать ввод. Вставить данные можно также с помощью сочетания клавиш CTRL+V или контекстного меню.

:::image type="content" source="media/service-quick-create-report/create-enter-data-window.png" alt-text="Ввод или вставка в окне &quot;Введите данные&quot;.":::

С помощью контекстного меню можно добавлять и удалять столбцы. Если вставленные данные содержат строку заголовка, выберите **Использовать первую строку в качестве заголовков** для автоматического преобразования первой строки в строку заголовка. Power BI обнаруживает типы данных автоматически, но их можно задать вручную. Нажмите кнопку **Тип данных** рядом с именем столбца. 

:::image type="content" source="media/service-quick-create-report/change-data-type.png" alt-text="Изменение типа данных столбца."::: 

В процессе создания Power BI автоматически создаст новый набор данных и сформирует их обобщенное представление. Эти автоматически сформированные визуальные элементы позволят максимально быстро превратить необработанные данные в полезные сведения.  

:::image type="content" source="media/service-quick-create-report/autogenerated-report.png" alt-text="Power BI автоматически создает визуальные элементы на основе данных.":::

Изменить данные, отображаемые в отчете, также очень просто. Воспользуйтесь панелью **Суммирование** для добавления полей в отчет или удаления их оттуда. Обновите данные о том, какие объекты необходимо измерить и проанализировать, установив или сняв соответствующие флажки. Power BI автоматически добавит или удалит диаграммы для отображения новых сочетаний.  

В настоящее время может отображаться не более трех мер, представленных в отчете в виде строк, и четырех категорий, представленных в нем в виде столбцов. 

:::image type="content" source="media/service-quick-create-report/summarize-pane.png" alt-text="Изменение способа обобщения данных в Power BI на панели &quot;Суммирование&quot;.":::

Чтобы обобщить поле по-другому, используйте его контекстное меню на панели **Суммирование** для переключения между суммой, средним, максимальным и минимальным значениями и т. д. 

:::image type="content" source="media/service-quick-create-report/summarize-options.png" alt-text="Переключение между суммой, средним, максимальным и минимальным значениями и т. д.":::

Этот отчет можно сохранить в рабочей области, в которой вы работали при первом выборе функции **Создать**. Оттуда вы можете поделиться им, как и любым другим отчетом. Когда в следующий раз вы или любые другие пользователи с разрешениями на редактирование откроете отчет, сразу же отобразятся эти возможности для быстрого редактирования.  

Если вы хотите перейти в режим полноценного редактирования, нажмите кнопку **Изменить** в строке меню. Имейте в виду, что после сохранения отчета в режиме полноценного редактирования вернуться к представлению быстрого редактирования невозможно.  

:::image type="content" source="media/service-quick-create-report/edit-entry-point.png" alt-text="Нажатие кнопки &quot;Изменить&quot; в строке меню.":::

Эти возможности упрощают создание отчетов на основе данных и предоставляют эффективные средства создания отчетов совершенно новой аудитории пользователей. Ознакомьтесь с новыми возможностями создания уже сегодня.

## <a name="considerations-and-limitations"></a>Рекомендации и ограничения

### <a name="licenses"></a>Лицензии

Для создания быстрого отчета необходима та же лицензия, что и для создания любого другого отчета в Power BI. Если у вас бесплатная лицензия Power BI, а вашей области "Моя рабочая область" назначены возможности "Премиум", быстрые отчеты можно создавать в ней. Если у вас есть лицензия Power BI Pro, быстрые отчеты можно создавать в любом месте службы Power BI. Подробные сведения см. на странице [Функции службы Power BI по типам лицензий](../fundamentals/service-features-license-type.md).


### <a name="get-data-limitations"></a>Ограничения на получение данных 

- В данный момент, если вы используете команду **вставки или ввода данных вручную**, обновить данные позже будет невозможно. Если вы хотите добавить, изменить или удалить данные позже, необходимо снова пройти рабочий процесс **создания** и получить новый отчет.  
- Если у вас есть файл CSV или Excel, для добавления данных необходимо использовать команду вставки. Команда отправки файла появится позже. 
- При копировании данных в окно **Введите данные** размер вставляемых данных не может превышать 512 КБ. 
- Имя таблицы не может содержать более 80 символов, а имена столбцов — более 512 символов.  
- Имена таблиц и столбцов не могут содержать двойные кавычки ("), точки (.), а также пробелы в начале или конце.  

### <a name="model-limitations"></a>Ограничения модели

При выполнении этого процесса создания вы создаете модель в интернет-версии в фоновом режиме. Эти создаваемые в интернет-версии модели похожи на те, которые создаются в Power BI Desktop, но имеют несколько ограничений.

| Компонент | Status  | Подробнее |
|---------|---------|---------|
| ALM | Сейчас не поддерживается | ALM не работает для наборов данных, созданных в веб-версии. |
| Создание собственных ключей (BYOK) | Сейчас не поддерживается | Вы не можете использовать собственный ключ шифрования для созданных в веб-версии наборов данных. |
| Приложения-шаблоны | Сейчас не поддерживается | Вы не можете создавать приложения для рабочих областей с наборами данных, созданными в веб-версии. |  
| Общедоступные API администрирования | Поддерживается частично | Поддерживается большинство общедоступных API-интерфейсов администрирования. Однако существует известная ошибка, связанная со следующей операцией: **Администрирование — группы: GetGroupsAsAdmin с развернутыми наборами данных**. Для этой операции создаваемые в веб-версии наборы данных возвращаются, но ContentProviderType неверно помечается как RealTime. |

### <a name="report-limitations"></a>Ограничения отчетов  

Если вы используете команду **Изменить** для переключения в режим полноценного редактирования и сохраните отчет, вы больше не сможете вернуться к автоматически созданному представлению с помощью панели "Суммирование". Power BI напомнит об этом при выборе команды **Изменить**.  

:::image type="content" source="media/service-quick-create-report/quick-create-switch-edit-mode.png" alt-text="Переключение в режим редактирования из режима быстрого создания.":::
