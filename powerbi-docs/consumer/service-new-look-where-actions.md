# <a name="the-new-look-where-did-the-actions-go"></a>Новый вид: куда переместились действия?

Служба Power BI (app.powerbi.com) имеет новое представление, которое упрощает просмотр отчетов и взаимодействие с ними. В результате многие действия в меню были перемещены. Если ваша организация использует новое представление, но вам удобнее работать со старым, эта статья поможет вам. Таблицы в этой статье сопоставляют старое расположение с новым. 

Ищете сведения о новом интерфейсе в **Power BI Desktop**? См. статью [Использование обновленной ленты в Power BI Desktop](../create-reports/desktop-ribbon.md).

|---------|---------|---------|
| Сохранить как | Файл > Сохранить как  | Файл > Сохранить копию |
| Печать | Файл > Печать | Файл > Напечатать эту страницу |
| Внедрить безопасным образом | Файл > Внедрить | Поделиться > Внедрить отчет > Веб-сайт или портал |
| Внедрить в SharePoint Online | Файл > Внедрить в SharePoint Online | Поделиться > Внедрить отчет > SharePoint Online |

| Экспорт в PowerPoint | Файл > Экспорт в PowerPoint | Экспорт > Экспорт в PowerPoint |
| Экспорт в PDF | Файл > Экспорт в PDF | Экспорт > PDF |
