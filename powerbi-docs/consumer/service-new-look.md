# <a name="the-new-look-of-the-power-bi-service"></a>"Новый внешний вид" службы Power BI

Служба Power BI (app.powerbi.com) имеет новый вид, который упрощает просмотр отчетов и взаимодействие с ними. Новый внешний вид стал проще и базируется на вашем знакомстве с другими продуктами Майкрософт. В рамках всей службы Power BI мы поместили содержимое отчета в центр внимания, переключившись на более светлую тему и обновив значки. 

Ищете сведения о новом интерфейсе в **Power BI Desktop**? См. статью [Использование обновленной ленты в Power BI Desktop](../create-reports/desktop-ribbon.md).

Ниже приведен обзор изменений в новом внешнем виде. Дополнительные сведения см. в нумерованных разделах:

![Снимок экрана: обзор изменений в новом внешнем виде.](media/service-new-look/power-bi-new-look-changes-callouts.png)

Ищете определенное действие? См. статью [Новый вид: куда переместились действия?](service-new-look-where-actions.md)

## <a name="1-vertical-list-of-pages"></a>1. Вертикальный список страниц 
В представлении "Чтение" имена страниц отчета отображаются в списке на вертикальной панели. Они выделены, их трудно пропустить, а их использование аналогично навигации в Word и PowerPoint. Вы можете увеличить или уменьшить оставшуюся часть области отчета: Измените размер вертикальной панели или щелкните двойную стрелку, чтобы полностью закрыть ее.

![Снимок экрана: имена страниц отчета в вертикальном расположении.](media/service-new-look/power-bi-new-look-report-pages.png)

Если у вас есть разрешение на изменение отчета, его можно настроить таким образом, чтобы имена страниц располагались в нижней части, как в режиме редактирования отчета. Дополнительные сведения см. в разделе [Настройка панели страниц](../create-reports/power-bi-report-settings.md#set-the-pages-pane) в статье "Изменение параметров для отчетов Power BI".

## <a name="2-simplified-action-bar"></a>2. Упрощенная панель действий 

Расположенная сверху обновленная панель действий содержит наиболее релевантные команды для потребителей отчетов. Вам будет проще экспортировать данные, обмениваться ими и общаться в Teams, а также настраивать подписку. 

![Снимок экрана: новая панель действий.](media/service-new-look/power-bi-new-look-action-bar.png)

## <a name="3-where-are-the-report-commands"></a>3. Где находятся команды отчетов?

Мы не удаляли никакие функциональные возможности из старого внешнего вида. Нажмите **Дополнительные параметры (...)** , чтобы открыть расширенные команды. См. статью [Новый вид: куда переместились действия?](service-new-look-where-actions.md).

## <a name="4-new-filter-experience"></a>4. Новые функции фильтров

По умолчанию в новом интерфейсе отображается новая панель "Фильтры" для отчета. При наведении указателя мыши на значок фильтра для визуального элемента отображаются все фильтры и срезы, влияющие на этот визуальный элемент.

![Снимок экрана: все фильтры и срезы, влияющие на этот визуальный элемент.](media/service-new-look/power-bi-new-look-filters.png)

## <a name="view-report-details"></a>Просмотр сведений об отчете 

Вы можете быстро ознакомиться со сведениями, такими как дата последнего обновления и контактные данные, прямо в верхнем баннере.  Откройте меню, чтобы просмотреть дополнительные сведения об отчете. Вы даже можете отправить владельцу отчета сообщение электронной почты.

![Снимок экрана: просмотр сведений отчета.](media/service-new-look/power-bi-new-look-metadata.png)

## <a name="no-changes-to-report-edit-mode"></a>Нет изменений в режиме правки отчета 

Мы сохранили возможности разработки в том виде, в котором они доступны в Power BI Desktop. Изменения нового внешнего вида касаются только режима чтения.

## <a name="dashboard-new-look-experience"></a>Интерфейс панели мониторинга с "новым внешним видом" 

Для панелей мониторинга упрощена панель действий. Это изменение аналогично тому, что было сделано для отчетов и приложений, но имеются функциональные различия. Ниже приведено пошаговое руководство по действиям на панели мониторинга.
 
![Снимок экрана: панель действий панели мониторинга в новом внешнем виде.](media/service-new-look/power-bi-dashboard-action-bar-new.png)
