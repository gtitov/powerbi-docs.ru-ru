# <a name="dark-mode"></a>Темный режим

Чтобы обеспечить вам максимально удобную работу, мобильное приложение Power BI для iOS и Android поддерживает светлый и темный режим отображения. В темном режиме яркость экрана значительно снижена, что упрощает просмотр содержимого.

![Сравнение темного и светлого режимов](media/mobile-apps-dark-mode/powerbi-mobile-darkmode-lightmode.png)

 В темном режиме весь интерфейс приложения отображается с темным фоном. Отображение другого содержимого Power BI при этом не меняется. Отчеты, панели мониторинга и приложения всегда отображаются в соответствии с задумкой дизайнера.
 
 По умолчанию мобильное приложение Power BI использует параметры устройства для определения режима экрана. Если устройство настроено для работы в темном режиме, приложение тоже включит темный режим.

>[!NOTE]
>Поддержка на уровне устройства для темного режима на устройствах Android официально доступна начиная с Android 10. На устройствах, работающих под управлением более ранних версий Android, темный режим в мобильном приложении Power BI можно установить с помощью параметров внешнего вида приложения.

Чтобы переключаться между светлым и темным режимами или разрешить параметрам устройства определять режим, перейдите на страницу **Параметры**, прокрутите вниз до пункта **Внешний вид** и коснитесь его, чтобы выбрать нужный режим.

![Параметры внешнего вида](media/mobile-apps-dark-mode/powerbi-mobile-appearance-settings.png)
