# <a name="use-the-power-bi-mobile-app-in-split-screen-mode"></a>Использование мобильного приложения Power BI в режиме разделения экрана

Применяется к:

| ![Планшет Android](./media/mobile-apps-split-screen/android-logo-40-px.png) |
|:--- |
| Планшеты Android |

Если ваш планшет Android поддерживает работу с несколькими окнами, вы можете просматривать свои данные в другом приложении и взаимодействовать с ним в режиме разделения экрана.

![Снимок экрана: Power BI в режиме разделения экрана.](media/mobile-apps-split-screen/power-bi-mobile-split-screen.png)

### <a name="using-split-screen-mode"></a>Использование режима разделения экрана

См. документацию по планшету, чтобы узнать, как открывать приложения в режиме разделения экрана на своем устройстве.

После открытия приложения в режиме разделения экрана вы сможете настроить размер экранов, перетащив разделительную линию между приложениями вправо или влево.

Режим разделения-экрана лучше подходит для отчетов, оптимизированных для просмотра на мобильных устройствах. Если вы являетесь автором отчетов, мы рекомендуем создавать отчет с оптимизированным для мобильных устройств макетом. Подробные сведения см. в статье [Оптимизация отчетов Power BI для мобильных приложений](../../create-reports/desktop-create-phone-report.md).

>[!IMPORTANT]
>Режим разделения экрана работает только на поддерживаемых планшетах Android. См. документацию устройства, чтобы узнать, поддерживает ли оно режим разделения экрана.
