# <a name="install-and-use-the-sample-sales-and-marketing-app-in-the-power-bi-service"></a>Установка и использование образца приложения "Маркетинг и продажи" в службе Power BI

[!INCLUDE[consumer-appliesto-yyny](../includes/consumer-appliesto-yyny.md)]

После знакомства с [общими принципами получения содержимого Power BI](end-user-app-view.md) давайте получим шаблон приложения "Маркетинг и продажи" с использованием Microsoft AppSource. Шаблон приложения "Маркетинг и продажи" включает пример данных, которые вы можете использовать для знакомства со службой Power BI. Это конкретное приложение содержит одну панель мониторинга и несколько страниц отчета. 

> [!IMPORTANT]
> Для установки этого приложения из Microsoft AppSource требуется лицензия **Power BI Pro**.  Дополнительные сведения см. в статье о [лицензиях](end-user-license.md).

## <a name="get-the-app-from-microsoft-appsource"></a>Получение приложения из Microsoft AppSource

1. Откройте [https://appsource.microsoft.com](https://appsource.microsoft.com).

   ![открытие сайта AppSource  ](./media/end-user-app-marketing/power-bi-appsource-start.png)

1. В поле поиска введите **пример Майкрософт** и выберите **Пример Майкрософт — "Маркетинг и продажи"** . 

    ![Получить приложения  ](./media/end-user-app-marketing/power-bi-appsource-search.png)

1. При необходимости изучите общие сведения и обзоры и просмотрите изображения.  Затем нажмите кнопку **Получить**.

   ![Предложение приложения в AppSource](./media/end-user-app-marketing/power-bi-app-offer.png)

1. Подтвердите, что хотите установить это приложение.

   ![Установить это приложение?](./media/end-user-app-marketing/power-bi-install-confirm.png)

5. После установки приложения в службе Power BI появится сообщение о ее успешном выполнении. Выберите **Перейти к приложению**. Или выберите приложение на своем холсте **Приложения**. В зависимости от того, как разработчик создал приложение, появится либо панель мониторинга приложения, либо отчет о приложении.


    ![Приложения в Power BI](./media/end-user-app-marketing/power-bi-marketing.png)

7.  Этот шаблон приложения настроен для открытия панели мониторинга. *Разработчик* приложений также мог настроить открытие приложения на одной из страниц отчета.  

    ![Снимок экрана: панель мониторинга для приложения.](./media/end-user-app-marketing/power-bi-dashboard.png)




## <a name="interact-with-the-dashboards-and-reports-in-the-app"></a>Взаимодействие с панелями мониторинга и отчетами в приложении
Изучите данные на панелях мониторинга и в отчетах, входящих в состав приложения. Вам доступны все стандартные возможности взаимодействия Power BI, такие как фильтрация, выделение, сортировка и детализация.  Все еще немного путаете панели мониторинга и отчеты?  Прочите [статью о панелях мониторинга](end-user-dashboards.md) и [статью об отчетах](end-user-reports.md).
